# Chocolatey Softwarepakete

## Build & Push
Mit PowerShell den folgenden Befehl starten.

```sh
choco pack .\<packetname>.nuspec
```

```sh
choco push .\<packetname>.nupkg --source https://push.chocolatey.org/
```
